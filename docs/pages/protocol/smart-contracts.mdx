## Core Contracts

The Aurum protocol is composed of several specialized Clarity smart contracts. All contracts are verifiable on the Stacks blockchain.

> **Note**  
> The protocol is currently on Testnet. Below are the deployed contract addresses. Mainnet addresses will be updated here post-launch.

### Testnet Addresses

#### Vault Contract
**Type:** `Address`  
`ST.../agbtc-vault-v1`  
Handles user deposits/withdrawals and agBTC minting/burning.

#### Token Contract
**Type:** `Address`  
`ST.../agbtc-token`  
The SIP-010 compliant fungible token contract for agBTC.

#### Strategy Registry
**Type:** `Address`  
`ST.../strategy-registry`  
Maintains the list of active whitelisted protocols (Zest, Hermetica).
